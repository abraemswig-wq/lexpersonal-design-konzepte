# LeXpersonal — 12 Design-Konzepte

Zwölf moderne Web-Design-Konzepte für die LeXpersonal GmbH (Personaldienstleistung, Münster — seit 2008). Alle Konzepte nutzen das echte CI: das markante X-Symbol, die Blautöne `#2778BC` (Royal) und `#009FE3` (Cyan), sowie Montserrat + Source Sans Pro als Hauptschriften.

**→ [Live-Übersicht aller 12 Konzepte](https://abraemswig-wq.github.io/lexpersonal-design-konzepte/)**

## Drei Gruppen

### B2B-Premium (Empfehlung für Konzern- & Industriekund:innen)
- **09 — Corporate Premium** — McKinsey/Deloitte-Anmutung
- **10 — Engineering Blueprint** — Technische Zeichnung
- **11 — Case Portfolio** — Pentagram-Stil mit Case-Studies
- **12 — Annual Report** — DAX-Geschäftsbericht

### Moderne Standards
- **01 — Editorial Serif Modern**
- **02 — Dark Tech / Gradient**
- **03 — Bento Modular**
- **04 — Swiss Brutalist**

### Statement-Designs
- **05 — Long-Form Scroll Story**
- **06 — Terminal / CLI**
- **07 — Memphis Maximalist**
- **08 — Newspaper / Print**
